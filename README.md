User activity heatmap for the LaborGit project

## Getting Started
package.json:
```json
{
    "dependencies": {
        "laborgit-user-heatmap": "^1.0.0"
    }
}
```
```shell
npm install
npm update
```

or

without package.json
```shell
npm install laborgit-user-heatmap
npm update laborgit-user-heatmap
```
